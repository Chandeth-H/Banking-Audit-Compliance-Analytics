# Banking-Audit-Compliance-Analytics

## Project Overview

This project is an enterprise-style Banking Audit & Compliance Analytics Dashboard developed in Power BI using simulated banking audit data. The dashboard is designed to help compliance teams, auditors, and management monitor operational risks, audit findings, fraud exposure, SLA breaches, and remediation effectiveness across branches and departments.

## Objectives
- Monitor audit findings and compliance issues across the organization
- Identify high-risk branches, departments, and fraud categories
- Track SLA breaches and remediation performance
- Analyze financial impact caused by audit findings
- Provide management-level risk intelligence through interactive dashboards
